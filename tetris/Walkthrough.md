# 俄羅斯方塊 開發完成總結

我們已經成功完成純網頁版本的俄羅斯方塊遊戲！所有的代碼都已存放於您的工作區 `c:\Users\richa\OneDrive\Documents\tetris` 資料夾中。

## 專案結構
- `index.html`: 遊戲主頁面，包含美觀的記分板、下一個方塊預覽以及按鍵說明。
- `style.css`: 採用深色美學設計 (Dark Mode Aesthetics)，方塊具有簡單的立體感與發光特效。
- `script.js`: 囊括所有的遊戲核心邏輯與前端 Web Audio API 音效。

## 完成的功能清單

### 1. 遊戲核心與視覺
- 支援標準 10x20 網格。
- 7 種 Tetromino，分別設定了鮮豔且不同的顏色。
- 側邊欄展示了遊戲分數 (Score)、消除行數 (Lines) 與 等級 (Level)。隨著消除行數增加，遊戲掉落速度會跟著變快。
- 動態預覽功能：能在右側小畫布看到「下一個方塊」。

### 2. 鍵盤控制與雙向旋轉
- **`←` / `→`**: 左右移動方塊。
- **`↓`**: 加速下落 (軟下落)。
- **`↑`**: 向右 (順時針) 旋轉。
- **`Z` 或 `z`**: 向左 (逆時針) 旋轉。
- **`Space` (空白鍵)**: 直接到底部並鎖定 (硬下落)。
> [!NOTE]
> 代碼內有針對空白鍵與方向鍵進行 `event.preventDefault()`，可以避免玩遊戲時不小心捲動網頁。

### 3. 音效體驗 (無額外資源檔)
我們實現了 `Web Audio API`，讓瀏覽器即時合成聲音：
- 移動 (`sounds.move`)、旋轉 (`sounds.rotate`) 會發出細微的短暫嗶聲。
- 硬下落 (`sounds.drop`) 會發出低沉的聲音。
- 消除行 (`sounds.clear`) 發出合成提示音。如果一次消除四行 (Tetris) 會有更高昂的和弦！
- Game Over 發出降調聲。
