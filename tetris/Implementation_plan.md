# 俄羅斯方塊 (Tetris) 實作計畫

本計畫將為您打造一個美觀、具備完整功能的純網頁版俄羅斯方塊遊戲。

## 專案架構
為了維持代碼的清晰與易於維護，我們將建立三個主要檔案，並放置於工作區 `c:\Users\richa\OneDrive\Documents\tetris` 目錄下：
1. `index.html`: 負責遊戲的版面結構（包含主遊戲區畫布、預覽區、分數面板）。
2. `style.css`: 負責視覺設計，包含深色主題、現代化字體與版面排版，讓遊戲看起來有質感。
3. `script.js`: 核心遊戲邏輯，包含方塊資料、控制、碰撞偵測、音效處理與遊戲迴圈。

## 遊戲功能與設計

### 1. 核心機制
- **版面大小**: 標準 10 欄 x 20 列的遊戲網格。
- **方塊 (Tetrominoes)**: 實作 7 種標準方塊 (I, J, L, O, S, T, Z)，並賦予不同的鮮豔顏色。
- **預覽方塊**: 遊戲畫面側邊將顯示下一個即將出現的方塊。

### 2. 控制方式
使用鍵盤進行操作：
- **左箭頭 (`ArrowLeft`)**: 向左移動。
- **右箭頭 (`ArrowRight`)**: 向右移動。
- **下箭頭 (`ArrowDown`)**: 軟下落（加速方塊下落）。
- **上箭頭 (`ArrowUp`)**: 順時針旋轉（向右旋轉）。
- **`Z` 鍵**: 逆時針旋轉（向左旋轉）。
- **空白鍵 (`Space`)**: 硬下落（方塊瞬間掉落至最底部並鎖定）。

### 3. 音效實作 (純前端)
為符合純前端且無須額外下載資源的要求，我們將使用瀏覽器內建的 **Web Audio API** 即時合成音效。
- **移動/旋轉音效**: 短促輕微的嗶聲。
- **硬下落/鎖定音效**: 較低沉的打擊聲。
- **消除行音效**: 響亮且有成就感的和弦提示音。
- **遊戲結束音效**: 降調的電子音。

### 4. 視覺體驗
- 使用 HTML5 `<canvas>` 進行渲染，確保效能流暢。
- **深色美學**: 背景使用深灰/黑色系，讓多色方塊更加凸顯。
- **光影與邊框**: 方塊渲染時會帶有簡單的邊框立體感。
